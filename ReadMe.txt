// Main life controller tool - REX

1. Inbox - хранилище записей без сортировки.
2. Next - список СКД из всех проектов.
3. Projects - картотека актуальных задач, объединенных общей целью.
4. Someday - список отложенных идей и невыполнимых задач.
5. Sheduled - события в календаре.
6. Reference - справочная информация.
7. Habits - привычки.
8. Rules - правила жизни.
9. Backlog - список курсов для прохождения, фильмов для просмотра, книг для прочтения.
10. Export - выгрузка данных в xls или txt.

// Routing / UX:
1. Главная страница - окно для добавления нового пункта (по умолчанию - в Inbox) и список последних СКД.
2. При переходе по каждой из ссылок в правом меню (кроме Export) выводится список пунктов содержимого.
3. При переходе по ссылке для Export выводится страница с настройками экспорта.
4. Пункт Next - собирает список СКД из всех проектов.
5. Пункт Projects - список названий проектов, у каждого есть описание, список СКД, категория и примерные сроки исполнения.
6. Someday - у каждого пункта есть название и описание.
7. Sheduled - вывод списка пунктов Next с датами исполнения.
8. Reference - каждый пункт имеет категорию и большое поле для описания.
9. Habits, Rules, Backlog : список пунктов без ограничения по содержимому.
10. Export - страница для экспорта всех данных из всех пунктов в файл или таблицу на экране.

// Вспомогательное по разработке:

После добавления новых классов - обновить автозагрузку композером в консоли : composer dump-autoload

Подключиться к mysql-серверу в консоли:
mysql -u root -p
Показать список БД:
show databases;
Выбрать БД:
use databasename;
Показать список таблиц:
show tables;
Очистить таблицу без удаления ее структуры:
TRUNCATE TABLE tablename;
